# Code of Conduct

MCP Home Manager is committed to being a welcoming, respectful project for everyone who participates in the community.

We expect contributors, maintainers, and community members to help create an environment where people can collaborate safely and constructively.

## Our standards

Examples of behavior that helps build a positive community include:

- Being respectful in disagreements and reviews
- Giving constructive, specific feedback
- Assuming good intent while still addressing real problems clearly
- Being patient with newcomers and people learning unfamiliar tools
- Respecting privacy, security disclosures, and personal boundaries

Examples of unacceptable behavior include:

- Harassment, discrimination, or hateful conduct of any kind
- Personal attacks, insults, trolling, or deliberately inflammatory behavior
- Sexualized language or unwanted sexual attention
- Publishing someone else's private information without explicit permission
- Repeatedly derailing discussions, spamming, or acting in bad faith
- Retaliating against someone for reporting a concern

## Scope

This code of conduct applies across project spaces, including:

- GitHub issues, pull requests, and code review
- Project documentation and discussions
- Any other public or private communication channels used for project coordination

It also applies when someone is representing MCP Home Manager in public spaces.

## Reporting concerns

If you experience or witness behavior that violates this code of conduct, please report it privately to the project maintainers instead of opening a public issue.

- Contact the repository owner or maintainers directly through GitHub
- Include links, screenshots, or other context that can help with review when possible
- If the report involves a maintainer, contact another maintainer if one is available

## Enforcement

Maintainers may take any action they believe is appropriate to protect the community and the project, including:

- Requesting a change in behavior
- Editing, hiding, locking, or removing content
- Rejecting contributions
- Temporarily or permanently restricting participation

Reports will be reviewed as promptly and fairly as possible. Maintainers will respect the privacy and safety of reporters to the extent practical.
